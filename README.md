# LENA — Dress Roomshow

تطبيق عربي RTL لإدارة محل فساتين المناسبات وتشغيل معرض واحد محلياً.

## الوظائف الحالية

- المخزون والعملاء والحجوزات والتسليم والاسترجاع.
- المدفوعات وتسوية العربون والغرامات والمصروفات والبيع المباشر.
- التقارير وإقفال اليومية وسجل التدقيق والنسخ الاحتياطي.
- منع تعارض الحجوزات وفترة تجهيز قابلة للضبط.

## التخزين

- الويب وPWA يستخدمان تخزيناً محلياً داخل المتصفح.
- نسخة Tauri تحفظ Snapshot داخل SQLite محلي في مجلد بيانات التطبيق وتعيد استعادته عند التشغيل.
- التطبيق Local-first لمعرض واحد حالياً، بدون مزامنة متعددة الأجهزة أو صلاحيات مستخدمين.

## التقنية

React وTypeScript وVite وTailwind CSS وReact Router وTauri 2 وSQLite.

## التحقق

- `npm test`
- `npm run lint`
- `npm run build`
- `npm run tauri -- info`

## حالة الإطلاق

الفرع الرئيسي يحتوي MVP تشغيلياً محلياً. أعمال Sale Readiness الإضافية تنفذ على فرع مراجعة منفصل قبل البيع النهائي.
