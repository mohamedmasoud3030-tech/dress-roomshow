Audit, stabilize, complete, and clean the current Dress roomshow app until it reaches a first usable v0.1 release for one dress showroom.

Core product scope:
- Dresses
- Customers
- Reservations
- Delivery / Return
- Payments
- Expenses

Customer → Dress → Reservation → Payment/Deposit → Delivery → Return → Fees/Refund → Dress status update → Dashboard/Reports.

PHASE 4 — IMPLEMENTATION PRIORITY
==================================================

Implement in this order.

P0. Real reservation workflow

P0. Delivery / Return workflow

- Record delivery condition
- Return from delivered reservation
